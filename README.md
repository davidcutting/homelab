# homelab-ansible
An Ansible playbook for bootstrapping my homelab

## What's it do

1. Set up k8s controller node
2. Set up k8s worker node
3. Install stuff necessary to get to desired cluster state. See add ons

2. Traefik Ingress
3. MetalLB LoadBalancer
